# Swarnaratna - Dry Fruit Website

## Description
Swarnaratna is a website dedicated to selling high-quality dry fruits. This project provides an online platform for customers to browse, select, and purchase a variety of dry fruits, including almonds, walnuts, raisins, and more.

## Features
- Product catalog with detailed descriptions and images
- Shopping cart functionality
- User-friendly interface for easy navigation
- Responsive design for mobile and desktop


## Installation
1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`


## Usage
Simply open the website in a browser and start browsing the dry fruit products. Add items to your cart and proceed to checkout.
